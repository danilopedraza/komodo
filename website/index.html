Why Symstatic?

Simplicity

While not aiming to be as simple as possible, Symstatic is a small and simple language. This allows it to be learned easily.

Agility

The purpose of Symstatic is getting fast to a correct solution, and see how things work together. At the cost of completeness, Symstatic allows you to get the “happy path” fast.

Familiarity

Symstatic is pretty unsurprising. If you know the patterns that Symstatic prefers, you will know how to use Symstatic.

Use Symstatic

Command-line tool

You can use the Symstatic interpreter through the CLI application. It includes a REPL!

Web playground

You can use Symstatic in your browser, with the playground.
